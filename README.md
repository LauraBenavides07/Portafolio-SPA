# 🌸 Portafolio Personal — Versión Bootstrap

> Sitio web personal desarrollado con Bootstrap 5 + CSS personalizado.

## 🛠️ Tecnologías

- **HTML5** — Estructura semántica
- **CSS3 + Bootstrap 5.3.0** — Grid responsivo
- **JavaScript (ES6+)** — Interactividad
- **Bootstrap Icons 1.10.5** — Íconos
- **Google Fonts** — Playfair Display + DM Sans

## ✨ Características

- ✅ Diseño 100% Responsive
- ✅ Validación de formulario en tiempo real
- ✅ Carrusel con swipe táctil
- ✅ Cursor personalizado
- ✅ Navbar colapsable en móvil

## 📁 Estructura

## 🚀 Cómo Ejecutar

1. Abre "index.html" en tu navegador
2. Requiere internet (CDN de Bootstrap)
